# 🐍 Snake Game (JavaScript)

A classic Snake Game built using **Vanilla JavaScript, HTML, and CSS**.

## 🚀 Features

* Smooth snake movement using grid-based logic
* Random food generation (avoids snake body)
* Score and High Score (stored using localStorage)
* Timer tracking
* Collision detection:

  * Wall collision
  * Self collision
* Game Win condition (fills entire grid)
* Restart functionality
* Head and body differentiation

## 🧠 Concepts Used

* DOM Manipulation
* Game Loop using `setInterval`
* Data Structures (Array as deque for snake movement)
* Hashmap-style grid (`blocks[x-y]` for O(1) access)
* Collision Detection Algorithms

## 🎮 Controls

* Arrow Keys to move the snake
